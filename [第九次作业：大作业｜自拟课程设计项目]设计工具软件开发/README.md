# 开发计划书

开发这个小游戏的灵感来源于一个名为duet的小游戏，整体预设目标如下

- 鼠标控制屏幕中心的一红一蓝两个小球旋转，左键左旋，右键右旋
- 屏幕中间自动生成障碍物并向下移动，我们需要控制小球躲避障碍物获取分数
- 随着小球不断地前进，分数会递增
- 如果触碰到障碍物会进行扣分
- 实时记录当前分数、最高得分和死亡次数

代码文件包含两个pde文件，最后实现效果见gif文件
